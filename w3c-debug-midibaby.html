If you already have connect MIDI devices when you open this web page, they will appear in the inputs/outputs lists but they will not receive any messages, because this application does not register any message handler for already connected devices.
Do the following:
  1. Disconnect the device.
  2. Reload the page.
  3. Reconnect the device.
  4. Wait for the device to appear on the inputs/outputs lists.
